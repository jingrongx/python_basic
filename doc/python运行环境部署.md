* 操作系统</br>
Ubuntu 18.04

* 安装Python3</br>
`apt instll python3`

* 查看python版本</br>
```
   ~ $ python3
   Python 3.6.7 (default, Oct 22 2018, 11:32:17) 
```

* 安装pip3</br>
`sudo apt-get install python3-pip`

* 安装虚拟环境</br>
`pip3 install virtualenv`

* 创建虚拟环境 .env</br>
`python3 -m virtualenv .env`

* 激活虚拟环境</br>
`source .env/bin/activate`

* 安装需要的python模块</br>
`pip install *`

* 退出虚拟环境</br>
`deactivate`
